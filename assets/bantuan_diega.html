Sebelum melakukan transaksi

Sebelum anda melakukan transaksi diwajibkan untuk mengisikan sisa saldo, pin dan memilih nomor server terlebih dahulu, apabila anda tidak melakukannya dengan benar maka transaksi tidak akan berhasil. Untuk melakukan hal itu anda perlu untuk membuka halaman Pengaturan pada aplikasi.

Bagaimana mengetahui sisa saldo

Untuk mengetahuinya anda cukup mengirim sms :

''SAL.PIN''

PIN adalah pin anda. Kemudian kirim sms tersebut ke nomor sms center Universal Reload.

Melihat transaksi yang sudah dilakukan

Untuk melihat informasi transaksi yang sudah anda lakukan, anda perlu untuk membuka halaman Catatan Transaksi pada aplikasi, kemudian pilih kategori catatan transaksi maka setelah itu anda mendapatkan informasi transaksi yang sudah anda lakukan. Tersedia juga fitur pencarian apabila anda memerlukannya.

Menghapus catatan transaksi

Perlu diketahui anda tidak dapat menghapus sebagian transaksi. Pengahapusan transaksi dilakukan untuk menghapus semua transaksi yang tercatat. Untuk dapat menghapus transaksi anda perlu untuk membuka halaman Catatan Transaksi dan mengklik tombol Hapus Catatan Transaksi.

Melakukan pengaturan aplikasi

Untuk mengatur aplikasi, anda dapat melakukannya pada halaman pengaturan yang dapat diakses dengan mengklik tombol Pengaturan pada menu utama. Pada halaman pengaturan anda dapat menyesuaian data saldo, pin dan memilih nomor server alternatif lainnya.

Apakah aplikasi ini menyimpan data pin pada server

Aplikasi ini tidak menyimpan data pin dan saldo pada server, data pada aplikasi tersimpan pada perangkat pengguna.

Memilih kartu SIM untuk mengirim SMS

Pemilihan kartu SIM yang digunakan untuk mengirim SMS dapat dilakukan pada pengaturan perangkat anda. Untuk Android versi Lolipop - Nougat dapat dilakukan dengan membuka pengaturan > SIM cards, lalu pilih kartu sim utama untuk melakukan pengiriman SMS
